# Python Project Template

Проект создан на основе шаблона с предустановленной проверкой качества кода.

## Особенности

- ✅ Автоматическая проверка flake8 при Pull Request
- ✅ Преднастроенные правила кодирования
- ✅ GitHub Actions workflow
- ✅ Исключены служебные папки (.venv, __pycache__ и др.)

## Быстрый старт

1. Нажмите "Use this template" для создания нового репозитория
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
